## HoardHog - Blazingly fast API for home inventory management

Manage your home inventory with HoardHog, easily host it yourself and never forget where your things are, where did you buy that item from and how many you have. Especially useful for homelabs and collectors.

### Features

 It doesn't do much just yet but for now:

* add, remove, modify, list a item.
* add, remove, modify, list a location.
